\doxysubsection{Detailed Description}
Describes a basic celestial body. 

Describes a celestial body in a N-\/\+Body system. Body is a point mass with defined initial coordinates and velocity. Regardless of name \doxylink{class_planet}{Planet} body may describe any point mass celestial body like planets and stars.
